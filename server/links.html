Источники

В качестве основного источника информации я планирую использовать следующие ресурсы:

Wikipedia - основные определения, необходимые для введения терминологии и общего знакомства

Blockgeeks - один из самых широкоизвестных англоязычных гайдов на эту тему

Bitmex - актуальный курс одной из основных валют - "биткоина"

Bitinfocharts - сайт очень большим количеством собранной за все время статистики, пригодится для формирования графиков

Lifehacker - простая и хорошо написанная статья, покрывающая бОльшую часть тем.

SergeyBreeze - финансовый блог одного из самых крупных трейдеров в России.

HighTech - издание, занимающееся различными прогнозами по этой теме.

GitHub - репозиторий со всем, что касается этой статьи.
